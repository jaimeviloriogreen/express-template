# Express template

### Installation command

- **Install modules**
`pnpm install`

- **Run server**
`pnpm run dev`

⚠️ You should install [pnpm](https://pnpm.io/installation) first, to use these commands

### Routes available
|Route|Description|
|--|--|
|`/`| **Home page** |
|`/contact`| **Contact page** |
|`/services`| **Services page** |
